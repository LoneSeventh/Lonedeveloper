What is Web Development?

  Web Site Development is a process of creating a new web site or implementing change to one already in use , e.g adding a significant new section to alive web site.

  In sample terms, the process represent a framework within which all activities-form inception to review ( and eventual demise, if necessary)- can take place.



There are 8 step in the development process. There are:

  • Planning:  Decide why you whant a web site and what to creat.

  • Content: Creat a list content you want.

  • Design:  Creat a design for displaying content.

  • Construction:  Write the code and load up the content.

  • Test:  Make sure everything work properly.

  • Hosting:  Choose a domain name and find the place in the internet to put your web site.

  • Publicity:  Build traffic via publicity a sitec.

  • Review:  Review a site at intervals it to make sure it succeeds.

  Of course web site development does not happen just for fun-it must initiated in some way.
As can been seen above, the development process takes place within the bound of business goals and User Needed. It is this the initiated and guide the course of planning ,design, content etc. Until you explore your goal and user , your website simply has no reason to exist.

  As such, the first step for creating a web site is to decide why you want it and who you are making for.
